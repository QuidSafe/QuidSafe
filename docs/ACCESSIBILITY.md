# QuidSafe — Accessibility Statement

**Last updated:** April 2026

---

## Our Commitment

QuidSafe is committed to making our app and website accessible to everyone, including people with disabilities. We aim to meet WCAG 2.1 Level AA standards across all platforms.

---

## What We Do

### Visual
- Minimum 4.5:1 colour contrast ratio for all text (WCAG AA)
- 7:1+ contrast in dark mode (WCAG AAA)
- Text resizes properly with system font size settings (Dynamic Type on iOS, font scaling on Android)
- No information conveyed by colour alone — icons and labels are always used alongside colour indicators
- Dark mode available to reduce eye strain

### Motor
- All interactive elements meet 44x44px minimum touch target size
- No gestures required that can't also be achieved with a simple tap
- No time-limited interactions (except HMRC submission which has generous deadlines)

### Screen Readers
- All screens are compatible with VoiceOver (iOS) and TalkBack (Android)
- All interactive elements have appropriate aria-labels
- All icons have descriptive labels for assistive technology
- Focus order follows visual reading order

### Cognitive
- Plain English throughout — no jargon, no technical tax language
- Clear, consistent navigation across all screens
- One action per screen where possible
- Error messages explain what went wrong and what to do next
- Loading states use skeleton screens (not spinners) to reduce anxiety

### Motion
- All animations respect `prefers-reduced-motion` system setting
- When reduced motion is enabled, transitions are replaced with instant state changes
- No auto-playing animations or videos

---

## Known Limitations

- Receipt scanning requires camera access and adequate lighting
- Some HMRC-generated content (linked from Tax Rules screen) may not meet our accessibility standards — this is outside our control
- PDF exports may have limited screen reader support (we are working on accessible HTML alternatives)

## Feedback

If you encounter any accessibility barriers while using QuidSafe:

- Email: accessibility@quidsafe.co.uk
- We aim to respond within 5 working days
- We will work with you to find a solution
